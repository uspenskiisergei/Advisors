# Advisors — Команда установки
# Введите в чат OpenWork:

```bash
git clone git@github.com:uspenskiisergei/Advisors.git .
```

Это склонирует проект Advisors в текущую папку (корень workspace).

После выполнения перезапустите OpenWork или напишите:
```
Перезагрузи проект
```

Затем начните сессию:
```
Хочу философскую сессию
```
