# Advisors — Система философского мышления

## Установка

### Шаг 1: Установите OpenWork

Скачайте с https://opencode.ai/download

### Шаг 2: Создайте папку для проекта

```bash
mkdir ~/Advisors
cd ~/Advisors
```

Или создайте папку в Finder в любом удобном месте.

### Шаг 3: Откройте папку в OpenWork

1. Запустите OpenWork
2. Нажмите **«Open Workspace»**
3. Выберите созданную папку

### Шаг 4: Введите команду установки

В чате OpenWork введите:

```
git clone git@github.com:uspenskiisergei/Advisors.git .
```

### Шаг 5: Готово!

Напишите агенту:

```
Хочу философскую сессию
```

---

## Быстрая установка (one-liner)

```bash
mkdir ~/Advisors && cd ~/Advisors && git clone git@github.com:uspenskiisergei/Advisors.git .
```

Затем откройте папку в OpenWork и напишите **«Хочу философскую сессию»**
