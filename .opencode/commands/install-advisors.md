# Установить Advisors

Клонирует проект Advisors в текущую папку.

## Команда

```bash
git clone git@github.com:uspenskiisergei/Advisors.git .
```

## Что делает

1. Клонирует репозиторий Advisors в корень workspace
2. Скачивает 11 агентов-мыслителей
3. Настраивает PARA-память
4. Подготавливает конфигурацию

## После установки

Напишите агенту:
```
Хочу философскую сессию
```
