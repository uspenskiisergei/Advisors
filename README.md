# Advisors (Советники)

Система философского мышления. 11 великих мыслителей в твоей команде.

Задаёшь вопрос — один из мыслителей отвечает с позиции своей философии. После каждой сессии — рефлексия и запись в память.

---

## 🚀 Установка

### Быстрый старт (One-liner)

```bash
curl -s https://raw.githubusercontent.com/uspenskiisergei/Advisors/main/install.sh | bash
```

### Ручная установка

```bash
# 1. Клонируйте репозиторий
git clone git@github.com:uspenskiisergei/Advisors.git ~/Advisors

# 2. Запустите установщик
cd ~/Advisors
chmod +x install.sh
./install.sh
```

Выберите **1) Установить и запустить** — скрипт автоматически:
- Установит OpenCode (если ещё не установлен)
- Скачает проект
- Запустит систему

### Требования

- macOS
- Git
- OpenCode (устанавливается автоматически)

## 📖 Использование

### Запуск

```bash
cd ~/Advisors
opencode .
```

### Начало сессии

Скажите агенту:

```
Хочу философскую сессию
```

Агент:
1. Загрузит инструкцию из Gist
2. Прочитает вашу память из `life/MEMORY.md`
3. Покажет контекст предыдущих сессий
4. Предложит выбрать мыслителя или задать вопрос

## 📝 PARA-система памяти

После каждой сессии агент автоматически:

1. Записывает ключевые события в `memory/YYYY-MM-DD.md`
2. Обновляет `life/MEMORY.md`:
   - Новые паттерны и сильные стороны
   - Ключевые инсайты
   - Историю сессий

## ❓ FAQ

**Q: Почему инструкция загружается из Gist?**
A: Чтобы вы могли обновить систему для всех пользователей, изменив один файл.

**Q: Могу ли я изменить агентов?**
A: Локальные агенты защищены. Но вы можете обновить инструкцию в Gist — она загрузится при следующей сессии.

**Q: Как работает память?**
A: `life/MEMORY.md` хранит контекст между сессиями. `memory/` — ежедневные заметки. Эти файлы не синхронизируются с Git.

**Q: Какая модель используется?**
A: `minimax-m2.5-free` — бесплатная модель от OpenCode.
