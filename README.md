# Advisors
Советники — 11 величайших мыслителей в твоей команде

Это система философского мышления. Работает просто: задаю вопрос — один из 11 мыслителей отвечает с позиции своей философии.

Кто в команде:
* Сократ — этика и диалог
* Платон — теория идей
* Аристотель — логика
* Конфуций — гармония
* Марк Аврелий — стоицизм
* Кант — мораль
* Декарт — мышление
* Адам Смит — экономика
* Маркс — материализм
* Ницше — воля к власти
* Эйнштейн — физика
  
Зачем это нужно:
Каждый мыслитель — это линза. Смотрю на вопрос через разные точки зрения и вижу то, что один не увидит.
После каждой сессии — рефлексия. Что услышал? Что резонирует? Записываю в долгосрочную память (PARA-система).

Как использовать:
Задаю вопрос → система выбирает мыслителя → отвечает от его лица → предлагает вопрос для размышления → сохраняю в память.
Это не про знания. Это про развитие мышления и самоосознания через диалог с великими умами.
