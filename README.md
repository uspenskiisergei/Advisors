# Advisors
Советники — 11 величайших мыслителей в твоей команде

Это система философского мышления. Работает просто: задаю вопрос — один из 11 мыслителей отвечает с позиции своей философии.

Кто в команде:
* Сократ — этика и диалог
* Платон — теория идей
* Аристотель — логика
* Конфуций — гармония
* Марк Аврелий — стоицизм
* Кант — мораль
* Декарт — мышление
* Адам Смит — экономика
* Маркс — материализм
* Ницше — воля к власти
* Эйнштейн — физика

Зачем это нужно:
Каждый мыслитель — это линза. Смотрю на вопрос через разные точки зрения и вижу то, что один не увидит.
После каждой сессии — рефлексия. Что услышал? Что резонирует? Записываю в долгосрочную память (PARA-система).

Как использовать:
Задаю вопрос → система выбирает мыслителя → отвечает от его лица → предлагает вопрос для размышления → сохраняю в память.
Это не про знания. Это про развитие мышления и самоосознания через диалог с великими умами.

## Быстрый старт

### 1. Установите OpenWork/OpenCode

Следуйте [официальной инструкции](https://opencode.ai/docs/getting-started)

### 2. Клонируйте репозиторий

```bash
git clone git@github.com:uspenskiisergei/Advisors.git
cd Advisors
```

### 3. Откройте в OpenCode

```bash
opencode .
```

### 4. Начните сессию

Скажите агенту: **«Хочу философскую сессию»**

## Как работает система

### Архитектура

```
┌──────────────────────────────────────┐
│         GitHub репозиторий           │
├──────────────────────────────────────┤
│  Файлы агентов → клонируются        │
│  Инструкция в Gist → загружается    │
│  Память пользователя → локально      │
└──────────────────────────────────────┘
```

### Структура папок

```
Advisors/
├── .opencode/
│   └── agents/           # Агенты-мыслители
│       └── philosophy.md  # Загрузчик инструкции из Gist
├── life/                  # PARA: граф знаний
├── memory/                # PARA: ежедневные заметки
├── opencode.jsonc          # Конфигурация
└── README.md
```

## Безопасность

### Защита от изменений

- **Агенты** — загружаются из Gist, не могут быть изменены локально
- **Инструкция** — обновляется только в Gist, автоматически применяется у всех
- **Файлы проекта** — защищены от случайного изменения

### Что пользователь может делать

- ✅ Использовать агентов для диалога
- ✅ Записывать в `memory/` и `life/`
- ✅ Обновлять инструкцию в Gist (если есть доступ)

### Что пользователю запрещено

- ❌ Изменять файлы `.opencode/agents/`
- ❌ Менять конфигурацию проекта
- ❌ Устанавливать новые плагины

## Обновление инструкции

Инструкция хранится в GitHub Gist и загружается автоматически.

**Ссылка:** https://gist.github.com/uspenskiisergei/eb9252e02eacd04c62a6ec5dd9cca9e2

Чтобы обновить инструкцию для всех пользователей:

1. Откройте Gist по ссылке выше
2. Измените содержимое `gistfile1.md`
3. Сохраните

Изменения автоматически применятся у всех пользователей при следующей сессии.
