# Advisors (Советники)

Система философского мышления. 11 великих мыслителей в твоей команде.

---

## 🚀 Установка

### 1. Установите OpenWork

Скачайте: https://opencode.ai/download

### 2. Создайте папку

```bash
mkdir ~/Advisors
cd ~/Advisors
```

Или вручную в Finder.

### 3. Откройте папку в OpenWork

1. Запустите OpenWork
2. Нажмите **«Open Workspace»**
3. Выберите созданную папку

### 4. Введите команду

В чате OpenWork:

```
git clone git@github.com:uspenskiisergei/Advisors.git .
```

### 5. Готово!

Скажите агенту:

```
Хочу философскую сессию
```

## 🔒 Безопасность

- ❌ Файлы агентов защищены (загружаются из Gist)
- ❌ Инструкция обновляется централизованно
- ✅ Ваши заметки в `memory/` и `life/` — локальные

---

## 📝 PARA-система памяти

После каждой сессии агент записывает:

- Ключевые события → `memory/YYYY-MM-DD.md`
- Паттерны и инсайты → `life/MEMORY.md`
