# Life — Система памяти PARA

Структура организации знаний и памяти по методу Tiago Forte.

## Папки

- `life/projects/` — активные проекты с целями и сроками
- `life/areas/` — области ответственности (люди, навыки, интересы)
- `life/resources/` — справочные материалы и темы
- `life/archives/` — архив завершённых проектов

## Использование

1. **Проекты** — создавай папку для каждого проекта
2. **Люди** — в `life/areas/people/<имя>/`
3. **Навыки** — в `life/areas/skills/<навык>/`
4. **Ежедневные заметки** — в `memory/YYYY-MM-DD.md`

## Для чего это нужно

PARA помогает:
- Сохранять контекст между сессиями
- Видеть прогресс во времени
- Систематизировать инсайты и решения
- Строить персональную базу знаний
